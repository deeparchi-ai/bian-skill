## 服务域关系示例

### 客户开户流程涉及的服务域

```
Customer Onboarding Flow:

1. Party Reference Data Directory
   └── 创建当事方记录

2. Customer Profile Management
   └── 创建客户档案

3. KYC/AML
   └── 身份验证和合规检查

4. Current Account
   └── 开立账户

5. Debit Card
   └── 发放借记卡

6. E-Branch Operations
   └── 开通网银
```

### 支付流程涉及的服务域

```
Payment Flow:

1. Current Account
   └── 检查余额

2. Payment Order
   └── 创建支付订单

3. Fraud Detection
   └── 欺诈检测

4. Payment Execution
   └── 执行支付

5. Correspondent Bank Operations
   └── 跨行清算（如需要）

6. Financial Accounting
   └── 记账
```
